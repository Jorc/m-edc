Questions Fréquentes

Le retour haptique ne fonctionne pas ?

  • Assurez-vous que votre appareil prend en charge le retour haptique (iPhone 7 et supérieur)
  • Vérifiez que Réglages → Sons et Haptiques → Haptiques Système est activé
  • Assurez-vous que votre téléphone n'est pas en mode silencieux (le mode silencieux peut désactiver les haptiques sur certains appareils)

Le son ne se joue pas ?

  • Vérifiez les paramètres de volume de votre appareil
  • Assurez-vous qu'aucun appareil Bluetooth n'est connecté (le son peut être acheminé vers les écouteurs)
  • Vérifiez si le son in-app est activé dans les Réglages

Comment changer les modes de curseur ?

Dans le menu déroulant en haut de l'interface principale, vous pouvez basculer entre 5 modes de curseur : Classique, Cadran, Grille, Arc de Bord, Arc du Pouce.

Comment changer les thèmes ?

Appuyez sur le bouton de réglages (icône d'engrenage) en haut à droite pour basculer entre 7 styles de thème : Néon, Minimal, Cyberpunk, Sakura, Pixel, Rétro, Verre Liquide (iOS 26+).

Comment utiliser l'exploration aléatoire ?

Appuyez sur le bouton dés pour générer aléatoirement de nouvelles séquences haptiques. Les utilisateurs gratuits peuvent explorer 3 fois par jour, les utilisateurs Pro ont un accès illimité. Les séquences découvertes peuvent être enregistrées dans vos favoris.

Comment enregistrer des séquences haptiques ?

Après avoir trouvé une séquence haptique que vous aimez dans l'exploration aléatoire, appuyez sur le bouton d'enregistrement (icône de téléchargement) pour l'enregistrer. Les utilisateurs gratuits peuvent enregistrer jusqu'à 3 séquences, les utilisateurs Pro ont un accès illimité.

Comment personnaliser les haptiques et les effets sonores ?

Appuyez sur le bouton de réglages en haut à droite → Haptiques Personnalisés (fonction Pro) pour créer et éditer vos propres séquences haptiques. Les utilisateurs Pro peuvent également personnaliser les effets sonores, en définissant différents sons pour chaque type d'haptique.

Comment partager et importer des séquences haptiques ?

Appuyez longuement sur une séquence enregistrée pour générer un code QR pour le partage. Scannez le code QR pour importer des séquences haptiques créées par d'autres utilisateurs (fonction Pro).

Comment ajuster la position et la taille du curseur ?

Appuyez sur le bouton de verrouillage en bas pour déverrouiller la mise en page du curseur. Après le déverrouillage, vous pouvez faire glisser la position du curseur et utiliser le pincement à deux doigts pour ajuster la taille. Appuyez à nouveau sur le bouton de verrouillage après l'ajustement pour enregistrer la position.

Quelles fonctionnalités a la version Pro ?

Après avoir déverrouillé Pro, vous obtenez les fonctionnalités suivantes :

  • Exploration aléatoire illimitée
  • Séquences haptiques enregistrées illimitées
  • Éditeur d'haptiques personnalisé
  • Conception sonore personnalisée
  • Importer/exporter des séquences haptiques
  • Accès à toutes les séquences haptiques prédéfinies

Comment restaurer les droits Pro après l'achat ?

Si vous changez d'appareil ou réinstallez l'application, vous pouvez appuyer sur le bouton "Restaurer les achats" dans les Réglages, et le système vérifiera automatiquement votre enregistrement d'achat Pro.

Exigences Système

  • iOS 15.0 ou supérieur
  • iPhone, iPad, iPod touch
  • Les appareils prenant en charge le retour haptique sont recommandés pour la meilleure expérience
